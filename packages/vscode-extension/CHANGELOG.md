# Changelog

All notable changes to the "Markdown Comment" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-01

### Changed

- Meta file format updated to ver2.0 (includes auto migration support)

### Added

- Client viewer app (Electron-based standalone viewer)

### Fixed

- Renderer issues with zero-length comments and header protection
- Various refactoring improvements

## [0.0.7] - Initial Release

### Added

- First stable release version
- Markdown preview integration with comment highlighting
- Sidebar synchronization between preview and comments
- Tree and Table views for comment management
- Comment threading and replies
- Status tracking (Open, Resolved, etc.)
- Custom tagging support
- Non-invasive storage (`.meta.jsonl` sidecar files)

## [0.0.1] - [0.0.6]

> **Warning**: These versions are deprecated and should not be used. They were pre-release development versions.
