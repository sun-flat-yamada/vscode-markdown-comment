# /code-review Command

Performs comprehensive code review for quality, security, and Clean Architecture compliance.

## Usage

```
/code-review [file or directory]
```

Without arguments, reviews recent uncommitted changes.

## Review Scope

- Clean Architecture compliance
- TypeScript best practices
- Test coverage
- Security considerations
- Documentation completeness

## Output

Structured review feedback with severity levels and fix recommendations.
